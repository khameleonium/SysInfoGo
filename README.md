# SysInfoGo

**SysInfoGo** — это кроссплатформенная утилита мониторинга системы, написанная на языке Go. 

Она предоставляет исчерпывающую информацию о состоянии вашего оборудования и операционной системы в различных форматах. Вы можете использовать её как классическую CLI-утилиту, интерактивный TUI-дашборд в терминале или запустить встроенный веб-сервер для мониторинга через браузер в реальном времени.

## Основные возможности
- **CPU & RAM**: Использование процессора и памяти, Load Average, кэш, swap.
- **GPU**: Поддержка мониторинга загрузки, видеопамяти и температуры видеокарт (NVIDIA/AMD).
- **Storage**: S.M.A.R.T. статус накопителей, логические разделы, свободное место, чтение/запись.
- **Network**: Сетевые интерфейсы, IPv4/IPv6, трафик, статистика скорости и история.
- **Processes**: Топ-10 процессов по нагрузке на CPU и RAM, возможность завершения зависших процессов.
- **Интерфейсы**:
  - `Текстовый (CLI)` — быстрый вывод статистики.
  - `TUI` — интерактивный терминальный интерфейс (`-t`, `--tui`).
  - `Web` — современный дашборд в браузере (`-w`, `--watch`).
- **Экспорт**: Поддержка сохранения отчетов в форматах JSON, CSV и HTML.
- **Локализация**: Полная поддержка русского языка и генерация пользовательских словарей (`--init-locale`).

## Скачивание и установка
Готовые бинарные файлы для **Windows**, **Linux** и **macOS** доступны в разделе [Releases](../../releases). Просто скачайте подходящий файл для вашей ОС и запустите в терминале.
